Class HighlightLayer

Generate a graphic layer to handle feature highlighting.

param

optional settings for the highlight layer layerId - id to use for the highlight layer. defaults to rv_highlight markerSymbol - esri symbol in server json format to symbolize the click marker. defaults to a red pin
